# vue seed 种子项目


### 一、项目结构
>
## src : 推荐项目结构
>
* assets ： 资源文件夹，图片、视频、文字等
>
* components ： 业务组件文件夹
>
* config ： 配置文件夹，通常配置根据环境不同配置不同 http server
>
* pages ：业务组件容器文件
>
* plgins ：自定义的基础文件
>
* store ：业务组件store文件
>
* router ：业务组件路由文件
>
* styles ：业务样式
>
* api ：业务数据请求
>
* index.html ：通用模板
* main.js ：业务主入口
* App.vue : 主路由入口
>
### 二、webpack配置
>
* webpack.config.js ： 通用webpack 配置   👇别名 整个框架中会用到，如果修改需要统一去修改 

### 三、运行，编译
> 
* 安装  npm i  
* 开发环境， npm start  
* 编译QA环境 npm run qa 
* 编译staging环境 npm run staging
* 编译prod环境 npm run prod
